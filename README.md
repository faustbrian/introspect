------

Runtime introspection helpers for PHP 8.4+ featuring trait detection, class hierarchy inspection, and reflection utilities.

## Requirements

> **Requires [PHP 8.4+](https://php.net/releases/)**

## Installation

```bash
composer require cline/introspect
```

## Documentation

- **[Getting Started](https://docs.cline.sh/introspect/getting-started)** - Installation and basic usage
- **[Fluent API](https://docs.cline.sh/introspect/fluent-api)** - Laravel-inspired chainable query builders
- **[Trait Inspection](https://docs.cline.sh/introspect/trait-inspection)** - Detect and query traits
- **[Class Hierarchy](https://docs.cline.sh/introspect/class-hierarchy)** - Inspect inheritance and interfaces
- **[Method Inspection](https://docs.cline.sh/introspect/method-inspection)** - Introspect class methods
- **[Property Inspection](https://docs.cline.sh/introspect/property-inspection)** - Inspect class properties
- **[Class Utilities](https://docs.cline.sh/introspect/class-utilities)** - Work with class names and namespaces
- **[Attribute Inspection](https://docs.cline.sh/introspect/attribute-inspection)** - PHP 8.0+ attribute support
